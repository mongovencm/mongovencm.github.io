# Skip List LeaderBoard

## Overview
This project implements a **skip list** data structure to maintain a dynamic soccer leaderboard. The program takes match results as input and keeps a sorted ranking based on:

- **Points**: 3 for a win, 1 for a tie, 0 for a loss
- **Goal Differential**: goals scored minus goals conceded
- **Goals Scored**: total goals scored 
- Alphabetical order if still tied after the above.

The leaderboard dynamically updates after every match that is input. It shows the teams in descending order based on ranking. It displays their number ranking, and it colors the top three teams as gold, silver, and bronze.
