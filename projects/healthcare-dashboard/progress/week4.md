# Week 4

- What did you do last week?

This past week I spent most my time finishing setting up and refining my GitHub website. Along with that, I began working more on the data wrangling and started writing some initial Python scripts to create cleaned versions of the raw datasets using pandas. 

- What do you plan to do this week?
  
This week I plan to continue developing scripts to clean and standardize the dataset formats so they can eventually be merged. After that, I will be able to begin creating initial visualizations and start exploring relationships in the data to determine what to include on the dashboard.

- Are there any impediments in your way?

The main challenge right now remains understanding how to properly structure and align multiple datasets that use different formats. 

- Reflection on the process you used last week, how can you make the process work better?

After last week's feedback, I’ve shifted toward a more incremental approach to data cleaning, focusing on smaller portions at a time and using Python scripts to generate processed versions while keeping the raw data unchanged. Since I am still in the middle of this process, it seems to be a more manageable approach, and I plan to continue refining it to reduce complexity and minimize the risk of introducing errors early in the pipeline.
