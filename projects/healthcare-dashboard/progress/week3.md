# Week 3

- What did you do last week?

This past week I set up my project environment and created a Github repository for it. I created a Python virtual environment to make sure dependencies are isolated and reproducible. I also started importing and exploring the athlete wellness data which include multiple CSV files. I used pandas to inspect their structure and found that there are some different formats used between some of the subjective data categories.

- What do you plan to do this week?
  
This week I plan to clean and standardize the dataset formats so they can be merged together. After that, I will begin building a unified master dataset and index it by player and date. Once the data is organized, I'll be able start creating initial visualizations and perform exploratory analysis to identify meaningful relationships.

- Are there any impediments in your way?

One challenge right now is understanding how to properly structure and align multiple datasets that use different formats. 

- Reflection on the process you used last week, how can you make the process work better?

This week I initially approached the dataset as something I would be able to analyze immediately, but quickly realized it will take some more time upfront to understand structure and inconsistencies. I plan to prioritize building a clean and reusable data pipeline before beginning analysis, which should make future modeling and visualization steps more efficient while reducing the need to repeatedly process raw data.
